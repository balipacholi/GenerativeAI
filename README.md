# GenerativeAI
GenAI-Lab: Building with Large Language Models
A curated collection of notebooks exploring the frontiers of Generative AI. This repository moves beyond static predictions to content creation, covering ANN (Artificial Neural Network), CNN (Convolution Neural Network), Prompt Engineering, RAG (Retrieval-Augmented Generation), and API integration with models like GPT-4 and Gemini.

Key Features:

ANN: Artificial Neural Network

CNN: Convolution Neural Network

Attention Mechanisms: Breaking down the "Attention is All You Need" paper with NumPy-based implementations.

Prompt Engineering: Techniques for zero-shot and few-shot learning.

RAG Pipelines: Connecting LLMs to private data using Vector Databases.

Fine-Tuning: Light-weight tuning of open-source models (Hugging Face).
